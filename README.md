# Luxury Watch Store

A premium MERN-style e-commerce application for high-end timepieces.

## Features
- **Premium UI**: Elegant black, gold, and white theme.
- **Full-Stack**: Express.js backend with SQLite (Better-SQLite3) for reliable local storage.
- **Authentication**: JWT-based secure login and registration.
- **Admin Dashboard**: Full CRUD for products and order management.
- **Responsive**: Optimized for all devices.
- **Animations**: Powered by Framer Motion.

## Setup Instructions
1. The application is pre-configured to run in this environment.
2. Backend runs on the same port as the frontend using Vite middleware.
3. Database is automatically initialized on first run.
